## Company Overview

Astera Labs is the only publicly traded pure-play CXL (Compute Express Link) memory expansion silicon company. The Leo CXL Smart Memory Controller, deployed in production on Microsoft Azure M-series VMs, enables hyperscalers to disaggregate GPU memory from compute, solving the GPU "memory wall" at scale. Company added to Nasdaq-100 on strong revenue acceleration.
