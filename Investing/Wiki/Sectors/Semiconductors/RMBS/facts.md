## Company Overview

Rambus is a pure-play memory interface IP and silicon company. The company licenses DDR5 RCD (Register Clock Driver) IP to every DRAM maker, capturing royalties on every memory module sold. HBM4E memory controller IP is the fastest in the industry. Zero fabrication capital requirements; high-margin licensing model.
