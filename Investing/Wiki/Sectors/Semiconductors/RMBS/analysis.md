# Rambus — Investment Analysis

## One-Line Thesis

Pure-play memory interface IP (DDR5, HBM4E); 40% RCD market share; zero fab capex, 84% margins, $14B TAM by 2035.

## Investment Thesis

| Date | Status | Notes |
|------|--------|-------|
| 2026-06-12 | Initial | Candidate for Memory Interface & Standards IP race |

Rambus occupies the "tollbooth" position in the memory interface layer. Every DDR5 RDIMM requires a Rambus RCD (Register Clock Driver) chip — a mandatory component for server DRAM. Every HBM4E system-on-module uses Rambus controller IP. As DDR5 adoption accelerates and HBM becomes the dominant memory interface for AI accelerators, Rambus collects recurring royalties on the entire installed base.

The TAM expansion is exceptional: memory interface chips were a $1.4B market in 2026, forecast to reach $14B by 2035 at a 29.5% CAGR. This growth is driven entirely by AI infrastructure — higher-speed memory, more memory modules per server, CXL memory pooling, and advanced packaging all require more sophisticated memory interface silicon.

Rambus's licensing model is highly defensive: with 40%+ DDR5 RCD market share and first-mover advantage in HBM4E controller IP, the company benefits from network effects (more deployed Rambus → more reference designs → faster ecosystem adoption). The 84% gross margin on products and 70%+ operating leverage allows rapid earnings expansion.

Bull case: TAM expansion from $1.4B to $14B + market share maintenance = 10x revenue potential over nine years, at 30%+ EBITDA margins. No fab exposure; all upside leveraged through IP licensing.

Risks: Customer diversification (reliance on Samsung, SK Hynix, Micron); competitor next-gen controller threat; memory cycle downturn.

**Key moat:** Installed base lock-in + first-mover HBM4E controller IP + royalty recurring revenue model.

**Key risks:** Memory cycle impact on DDR5 RCD volume, customer in-house controller development, IP litigation.
